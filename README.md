# Product CRUD API

A REST API built with Spring Boot and MongoDB Atlas for managing products.

## Tech Stack
- Java 17
- Spring Boot 3.4.1
- Spring Data MongoDB
- Maven
- Lombok

## Prerequisites
- Java 17+
- Maven
- MongoDB Atlas account

## Setup & Configuration

1. Clone the repository
   git clone <your-repo-url>
   cd Assignment

2. Update `src/main/resources/application.properties` with your MongoDB URI:
   spring.data.mongodb.uri=mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/productdb?appName=Cluster0
   spring.data.mongodb.database=productdb

3. Run the application:
   mvn spring-boot:run

App will start on http://localhost:8080

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/products | Create a product |
| GET | /api/products | Get all products |
| GET | /api/products/{id} | Get product by ID |
| PUT | /api/products/{id} | Update a product |
| DELETE | /api/products/{id} | Delete a product |

## Sample Request Body (POST/PUT)

{
"name": "Laptop",
"category": "Electronics",
"price": 75000.0,
"stock": 10
}

## Database Design

Collection: products
Fields:
- id (String) - Auto generated by MongoDB
- name (String) - Product name
- category (String) - Product category
- price (Double) - Product price
- stock (Integer) - Available stock
